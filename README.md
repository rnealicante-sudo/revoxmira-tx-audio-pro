# revox TX - High Fidelity Audio Engine

Una herramienta de transmisión de audio de alta calidad inspirada en VDO.Ninja, con control de ganancia y vúmetros en tiempo real.

## Características

- **Calidad Profesional**: Configuración optimizada para audio de alta fidelidad (510kbps, Pro Audio).
- **Control de Ganancia**: Ajuste fino de la entrada de audio mediante faders de hardware simulados.
- **Monitoreo Visual**: Vúmetros detallados con indicadores de pico.
- **Integración VDO.Ninja**: Genera enlaces de transmisión WebRTC seguros y eficientes.
- **Compartir Fácil**: Botones integrados para WhatsApp, Telegram y Email.

## Instalación

```bash
npm install
npm run dev
```

## Uso

1. Selecciona tu fuente de entrada de audio.
2. Pulsa el botón de encendido.
3. Ajusta la ganancia para que el nivel se mantenga en el área amarilla/verde.
4. Pulsa "Invite Link" para compartir el enlace con tus auditores.
5. Pulsa "Launch Audio Engine" para iniciar tu propia transmisión en VDO.Ninja.

Diseñado con una estética de hardware técnico y precisión digital.
